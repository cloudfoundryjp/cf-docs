---
title: Cloud Foundryの使い方
---

Cloud Foundry は Open PaaSであり、開発フレームワークとアプリケーションサービスを提供します。Cloud Foundry は、素早く簡単にアプリケーションのビルド、テスト、デプロイ、スケールが出来ます。

このセクションでは、開発者がアプリケーションをCloud Foundryにデプロイする方法を説明します。
