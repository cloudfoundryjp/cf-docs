Install Ruby via rvm

This document describes how to install Ruby using rvm.

Note : Assuming you are starting with a pristine Ubuntu 12.10 VM or a Physical Machine

  1. Install curl using apt-get

    sudo apt-get install curl

  2. Install rvm using curl

    \curl -L https://get.rvm.io | bash -s

  3. Reload the .bash_profile in the current terminal

    source ~/.bash_profile

    Note : To make sure bash is run as a login shell and rvm is loaded, please follow the steps in the doc Integrating RVM with gnome-terminal

  4. Check the required dependencies for installing ruby

    rvm requirements

  5. Install the required pre-reqs for installing ruby

    sudo /usr/bin/apt-get install build-essential bison openssl libreadline6 libreadline6-dev curl git-core zlib1g \
  zlib1g-dev libssl-dev libyaml-dev libsqlite3-0 libsqlite3-dev sqlite3 libxml2-dev libxslt-dev autoconf \ 
  libc6-dev ncurses-dev

  6. Install ruby 1.9.2 using rvm

    rvm install 1.9.2

  7. Set the ruby version to 1.9.2

    rvm use 1.9.2

  8. Check that the correct version of ruby has been set

    ruby -v

  9. Install bundler gem

    gem install bundler

  10. Install rake gem

    gem install rake
